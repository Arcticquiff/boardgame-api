### ✨ [Demo](https://tp-boardgame-api.herokuapp.com/)

## Setup

You will need to run the following commands:

```sh
npm i
npm i jest -D
npm i supertest -D
npm i express
npm i pg
npm i pg-format
npm i dotenv
```
Then create two .env files inside the envfiles folder with the following inside:

```sh
PGDATABASE=your_database_here
```

## Run tests

```sh
npm t
```
